# Contributing

Thank you for helping improve Spec-Driven Agent Fluency.

## Good Contributions

- Correct inaccurate engineering guidance.
- Improve clarity for non-native English speakers.
- Add realistic specification, validation, or review drills.
- Strengthen source attribution.
- Add evidence-based learning exercises.
- Report inaccessible or culturally ambiguous language.

## Contribution Requirements

1. Write original material in your own words.
2. Do not copy proprietary course lessons, books, graphics, or paid content.
3. Cite authoritative sources for factual or time-sensitive claims.
4. Separate established evidence from opinion or inference.
5. Avoid medical promises about cognition, memory, or neuroplasticity.
6. Keep examples vendor-neutral unless vendor behavior is the subject.
7. Use plain, professional English and define specialized terms.

## Exercise Format

Prefer this structure:

```text
Objective
Core concepts
Workflow
Vocabulary
Production-oriented example
Common failure
Recall drill
```

## Pull Requests

- Keep each pull request focused on one coherent improvement.
- Explain the learner problem being solved.
- Identify sources used.
- Confirm that the contribution is original or properly licensed.
- Check headings, links, spelling, and Markdown rendering.
